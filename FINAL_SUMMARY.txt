================================================================================
                           SAFETY CHECKS
================================================================================

Six Critical Safety Validations:
  
  ✓ No images found           → Raises ValueError with message
  ✓ Data path missing         → Raises FileNotFoundError
  ✓ ICP < 50 points           → Skips ICP for that frame
  ✓ All clustering noise      → Prints warning, continues
  ✓ Voxel size = 0            → Clamps to epsilon (1e-6)
  ✓ Registration single frame → Skips with [SKIP] message

All safety checks provide context and don't break execution.
